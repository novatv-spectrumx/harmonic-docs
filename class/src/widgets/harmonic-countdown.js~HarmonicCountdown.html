    HarmonicCountdown

    Extends:

    HarmonicWidget → HarmonicCountdown

    The HarmonicCountdown widget provides a countdown text field that can be controlled with the widget controls.

    This class implements the Harmonic Countdown Widget. It derives from the HarmonicWidget class and implements necessary functions to meet the needs of this widget.

    The HarmonicCountdown widget requires that the registered createdjs object have the following structure:

    • The createjsObj must be a createjs.MovieClip named correctly (reference HarmonicWidget).
    • The createjs.MovieClip must contain a createjs.Text child named 'harmonicField'. This text object represents the dynamic text field.

    This class defines functions for:

    • updateTextField - converts the text to a number and updates the countdown
    • restartFieldAnimation - restarts the countdown
    • stopFieldAnimation - stops the countdown

    To use this class, simply register the createjs object using the static register function. The created HarmonicCountdown object is returned from the register function. If there is a error, then 'undefined' is returned and an error will be logged.

    The 'countdownFrom' can be specified in the register function or it can be set using the 'countdownFrom' property on the object returned from the call to register.

    When the countdown reaches zero, the widget will dispatch the HarmonicCountdown.COUNTDOWN_COMPLETE_EVENT event on the document.

    The manualControl property allows the countdown to manually controlled independently from the start/stop field animation controls. This may be useful for countdown timers that need to start in the middle of an animation.

    Example:

    1. When the createjs object is registered with the class in the first frame of the movie clip:
let countdown = HarmonicCountdown.register(this, 20, false);  // set countdown to 20 seconds with no manual control

2. Or set the 'countdownFrom' property from the returned object:
let countdown = HarmonicCountdown.register(this);
countdown.countdownFrom = 20;

document.addEventListener(HarmonicCountdown.COUNTDOWN_COMPLETE_EVENT, (event) => {
    hjs.info("The countdown has completed");
});
