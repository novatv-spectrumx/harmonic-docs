    HarmonicWidget

    Direct Subclass:

    HarmonicCountdown, HarmonicLabel, HarmonicLabelScroll, HarmonicMovieClip, HarmonicTextField, HarmonicTextScroll, HarmonicVideo, HarmonicXMLHttpRequest

    This class is the base class for all Harmonic Widgets. All widgets should be derived from this class.

    During construction, this class will:

    1. Add a listener to the document for the WIDGET_EVENT_<fieldNum> event. The listener function is called '_onWidgetEvent'. This function will handle all communication from the Harmonic Template. It is the responsibility of the derived class to implement the functions that they need. Reference the list below.
    2. Dispatch the WIDGET_CREATED event. This event will be received by the Harmonic Template. The Harmonic Template will maintain a list of all created widgets.

    Functions that should be overridden by derived classes if needed:

    • updateTextField
    • appendTextField
    • updateImageField
    • renderField
    • restartFieldAnimation
    • stopFieldAnimation

    In Adobe Animate, a widget is a createjs.MovieClip. The naming convention for widgets is 'field<fieldNum>_<name>'. Where <fieldNum> is a number from 0-254 and uniquely identifies the widget. <name> is a string that will be included in the Template Description.

    Since a createjs.MovieClip is a container, it can contain children. Widgets should validate that they contain their required children before allowing the register function to succeed. Reference each widget to better understand their required children.

    public static validate(derivedClass: String, createjsObj: Object, containsTextField: Boolean): Object source

    Validates that the createjs object is structured properly. Widgets should be composed of a createjs MovieClip and named correctly. Also, if the MovieClip contains a TextField, it too must be named correctly.

    Proper naming:

    • MovieClip name: 'field<fieldNum>_<name>'
    • TextField name: harmonicField

    Params:

    NameTypeAttributeDescription
    derivedClass String

    the name of the derived class. This will be used for logging.
    createjsObj Object

    the createjs object
    containsTextField Boolean

    indicates if the MovieClip contains a TextField to be validated

    Return:

    Object

    returns an object if validation passes; otherwise, undefined is returned
