    HarmonicMovieClip

    Extends:

    HarmonicWidget → HarmonicMovieClip

    The HarmonicMovieClip widget provides an easy way to control a createjs.MovieClip with the widget controls.

    This class implements the Harmonic HarmonicMovieClip Widget. It derives from the HarmonicWidget class and implements necessary functions to meet the needs of this widget.

    The HarmonicMovieClip widget requires that the registered createdjs object have the following structure:

    • The createjsObj must be a createjs.MovieClip named correctly (reference HarmonicWidget).

    This class defines functions for:

    • restartFieldAnimation - restarts the animation
    • stopFieldAnimation - stops the animation

    To use this class, simply register the createjs object using the static register function. The created HarmonicMovieClip object is returned from the register function. If there is a error, then 'undefined' is returned and an error will be logged.

    Example:

    let movieClip = HarmonicMovieClip.register(this);
movieClip.stopFieldAnimation(true);
