    HarmonicLiveAVO

    The HarmonicLiveAVO widget provides control of the system's live audio voice over unit (AVO).

    The HarmonicLiveAVO widget requires a single createjs.MovieClip to be registered. This widget is defined in the Harmonic Template Widget Library. It just needs to be included in the template for it to work.
