    HarmonicTextField

    Extends:

    HarmonicWidget → HarmonicTextField

    The HarmonicTextField widget provides a dynamic text field that can be controlled with the widget controls.

    This class implements the Harmonic TextField Widget. It derives from the HarmonicWidget class and implements necessary functions to meet the needs of this widget.

    The HarmonicTextField widget requires that the registered createdjs object have the following structure:

    • The createjsObj must be a createjs.MovieClip named correctly (reference HarmonicWidget).
    • The createjs.MovieClip must contain a createjs.Text child named 'harmonicField'. This text object represents the dynamic text field.

    This class defines functions for:

    • updateTextField - updates and renders the dynamic text field
    • appendTextField - appends text to the dynamic text field and renders
    • renderField - renders the text field

    To use this class, simply register the createjs object using the static register function. The created HarmonicTextField object is returned from the register function. If there is a error, then 'undefined' is returned and an error will be logged.

    Example:

    let textfield = HarmonicTextField.register(this);
textfield.updateTextField("updated text", true);
