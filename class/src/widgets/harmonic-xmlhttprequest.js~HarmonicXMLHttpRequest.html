    HarmonicXMLHttpRequest

    Extends:

    HarmonicWidget → HarmonicXMLHttpRequest

    The HarmonicXMLHttpRequest widget provides a mechanism to read data from a webserver. It contains a dynamic text field that can be used to load the URL.

    This class implements the Harmonic XMLHttpRequest Widget. It derives from the HarmonicWidget class and implements necessary functions to meet the needs of this widget.

    The XMLHttpRequest widget requires that the registered createdjs object have the following structure:

    • The createjsObj must be a createjs.MovieClip named correctly (reference HarmonicWidget).
    • The createjs.MovieClip must contain a createjs.Text child named 'harmonicField'. This text object represents the dynamic text field.

    This class defines a function for:

    • updateTextField - the text defines the URL to load from

    To use this class, simply register the createjs object using the static register function. The created HarmonicXMLHttpRequest object is returned from the register function. If there is a error, then 'undefined' is returned and an error will be logged.

    The 'url' can be specified in the register function or it can be set using the 'url' property on the object returned from the call to register.

    After a load is request, the widget will dispatch the HarmonicXMLHttpRequest.XML_HTTP_REQUEST_COMPLETE_EVENT event on the document or the HarmonicXMLHttpRequest.XML_HTTP_REQUEST_TIMEOUT_EVENT event if the asynchronous request has timed out

    Example:

    1. When the createjs object is registered with the class in the first frame of the movie clip:
let xhr = HarmonicXMLHttpRequest.register(this, "test.xml");

2. Or set the 'url' property from the returned object:
let xhr = HarmonicXMLHttpRequest.register(this);
xhr.url = "test.xml";

document.addEventListener(HarmonicXMLHttpRequest.XML_HTTP_REQUEST_COMPLETE_EVENT, (event) => {
    hjs.info("Your data is loaded: " + event.detail);
});

document.addEventListener(HarmonicXMLHttpRequest.XML_HTTP_REQUEST_TIMEOUT_EVENT, (event) => {
    hjs.error("Your load timed out");
});

xhr.load();
