    HJS

    This class provides the interface to/from the HTML renderer. It uses a websocket connection to send and receive data from the HTML renderer.

    This class provides the following capabilities:

    • on platform vs. desktop detection
    • log message functionality (to console and system log)
    • individual interfaces for sending information to the HTML renderer
    • dispatches COMMAND_RECEIVED_EVENT when valid data is received from the HTML renderer
